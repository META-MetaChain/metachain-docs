Here, we offer a compilation of tools and blockchain explorers that enable you to examine and oversee transactions, smart contracts, and overall blockchain activity related to decentralized applications (dApps) on different metachain chains.

| Tool       | Use-cases                                                           | Relevant links                                                                                                                                                                                                                                                                                                                      |
| ---------- | ------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| metachain   | Track/trace transactions and examine addresses on metachain networks | <ul><li>metachain One: [https://metachain-i.co/](https://metachain-i.co/)</li><li>metachain Nova: [https://nova.metachain-i.co/](https://nova.metachain-i.co/)</li><li>metachain Sepolia: [https://sepolia.metachain-i.co](https://sepolia.metachain-i.co)</li><li>metachain Goerli: [https://goerli.metachain-i.co/](https://goerli.metachain-i.co/)</li></ul> |
| DexGuru    | Track/trace transactions and examine addresses on metachain networks | <ul><li>metachain One: [https://metachain.dex.guru/](https://metachain.dex.guru/)</li><li>metachain Nova: [https://nova.dex.guru/](https://nova.dex.guru/)</li></ul>                                                                                                                                                                    |
| Blockscout | Track/trace transactions and examine addresses on metachain networks | <ul><li>Stylus Testnet: [https://stylus-testnet-explorer.metachain-i.co](https://stylus-testnet-explorer.metachain-i.co)</li></ul>                                                                                                                                                                                                        |
| Dune       | Visualize and analyze metachain network data                         | <ul><li>[Dune](https://dune.com/)</li><li>[metachain community-created Duune dashboard](https://dune.com/browse/dashboards?q=metachain)</li></ul>                                                                                                                                                                                     |
